# Triage Labels

相关 skills 使用五个标准 triage 角色描述 issue 状态。本文将这些角色映射到本仓库 issue tracker 中实际使用的 label 字符串。

| mattpocock/skills 中的角色 | 本仓库 label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可以交给 agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会继续处理 |

当 skill 提到某个角色时，使用上表对应的 label 字符串。

如果仓库未来采用不同的 label 体系，请只修改右侧“本仓库 label”列。
