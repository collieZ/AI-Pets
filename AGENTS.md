# AI-Pets Agent 指南

## 通用约定

本项目的沟通、文档、注释说明和面向用户的输出默认优先使用中文。只有在 API 名称、协议字段、命令、错误信息、第三方术语或英文原文更准确时，才保留英文。

## Agent skills

### Issue tracker

本仓库 `collieZ/AI-Pets` 使用 GitHub Issues 跟踪任务；外部 PR 不作为 triage 请求入口。详见 `docs/agents/issue-tracker.md`。

### Triage labels

使用默认五标签 triage 词汇：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。详见 `docs/agents/triage-labels.md`。

### Domain docs

本仓库使用 single-context 领域文档布局：根目录 `CONTEXT.md` 和 `docs/adr/`。详见 `docs/agents/domain.md`。
